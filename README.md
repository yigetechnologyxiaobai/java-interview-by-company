## 🎯 各厂面试特点

| 公司 | 面试风格 | 重点关注 |
|------|---------|---------|
| **阿里** | 深挖底层原理 | JVM、并发、分布式、MySQL |
| **腾讯** | 注重基础扎实 | 算法、网络、操作系统、基础 |
| **字节** | 算法驱动 | 算法、系统设计、基础 |
| **美团** | 业务场景结合 | 中间件、分布式、高并发 |
| **京东** | 架构能力 | 并发、MySQL、架构设计 |
| **华为** | 综合评估 | 基础、项目、综合素质 |
| **百度** | 算法导向 | 算法、基础、搜索技术 |
| **快手** | 高并发实战 | 缓存、消息队列、性能优化 |
| **小红书** | 中间件深度 | Redis、MySQL、分布式 |
| **滴滴** | 高可用设计 | 分布式、容灾、架构 |
| **网易** | 基础扎实 | Java基础、并发、JVM |
| **拼多多** | 高压高效 | 高并发、数据库、架构 |

## 📖 使用说明

1. **按公司准备**：针对目标公司重点准备对应章节
2. **横向对比**：同一知识点，看不同公司的问法
3. **查漏补缺**：对照高频题库，补强薄弱环节
