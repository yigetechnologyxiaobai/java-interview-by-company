## 携程简介

携程是中国领先的在线旅行服务公司，业务涵盖机票、酒店、度假、商旅等，技术团队在微服务架构、高可用系统建设方面有深厚积累。

## 面试特点

- **考察重点**：微服务治理、高可用架构、MySQL、Redis、消息队列
- **面试流程**：一二面技术面 + 三面主管面 + HR面
- **难度评估**：中等偏上，三面常有场景设计题
- **风格特色**：偏重业务场景落地，关注候选人对系统可用性的理解
