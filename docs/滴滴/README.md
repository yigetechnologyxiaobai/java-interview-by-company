# 滴滴 面试题

## 面试风格

**特点**：注重业务场景，关注高并发和分布式，喜欢项目经验

**轮次**：通常 2-3 轮技术面 + 1 轮 HR

**重点**：分布式、MySQL、Redis、消息队列、业务场景设计

**面试官画像**：喜欢问业务场景，关注架构设计能力

## 面试技巧

1. **重视业务**：滴滴业务场景重要，准备派单、订单等场景
2. **项目经验**：项目难点要能说清楚
3. **技术广度**：分布式、MySQL、Redis 都要懂
4. **系统设计**：关注架构设计能力
