## 分布式高频题


### Q10: 分布式系统概念？

**核心概念**：

分布式系统是多台计算机协同工作，对外表现为一个系统。

**CAP 定理**：

```
┌─────────────────────────────────────────────────────────────────┐
│                       CAP 定理                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  C - 一致性（Consistency）                                       │
│     └─ 所有节点在同一时间看到相同数据                           │
│                                                                 │
│  A - 可用性（Availability）                                      │
│     └─ 每个请求都能得到响应（成功或失败）                       │
│                                                                 │
│  P - 分区容错性（Partition Tolerance）                           │
│     └─ 网络分区时，系统仍能运行                                 │
│                                                                 │
│  定理：最多只能同时满足两个                                      │
│  - CP：一致性 + 分区容错（Zookeeper）                           │
│  - AP：可用性 + 分区容错（Eureka）                              │
│  - CA：一致性 + 可用性（单机系统，无意义）                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**微服务架构**：

```
┌─────────────────────────────────────────────────────────────────┐
│                     微服务架构                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  服务网关（Gateway）                                             │
│     └─ 路由、限流、鉴权                                         │
│                                                                 │
│  服务注册与发现（Nacos/Eureka）                                  │
│     └─ 服务注册、服务发现                                       │
│                                                                 │
│  配置中心（Nacos/Spring Cloud Config）                          │
│     └─ 统一配置管理                                             │
│                                                                 │
│  服务调用（Feign/Dubbo）                                        │
│     └─ 声明式调用                                               │
│                                                                 │
│  负载均衡（Ribbon）                                              │
│     └─ 客户端负载均衡                                           │
│                                                                 │
│  熔断降级（Sentinel/Hystrix）                                    │
│     └─ 服务保护                                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
