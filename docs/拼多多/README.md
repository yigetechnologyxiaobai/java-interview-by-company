## 六、设计模式


### Q11: 项目中使用的设计模式？

**常见模式**：

| 模式 | 应用场景 |
|------|---------|
| 单例模式 | 配置类、连接池 |
| 工厂模式 | Bean 创建 |
| 策略模式 | 支付方式选择 |
| 模板方法 | 业务流程模板 |
| 责任链模式 | 过滤器、拦截器 |
| 观察者模式 | 事件监听 |
| 代理模式 | AOP、RPC |

**示例**：

```java
// 策略模式：支付方式
public interface PaymentStrategy {
    void pay(int amount);
}

public class AlipayStrategy implements PaymentStrategy {
    public void pay(int amount) {
        System.out.println("支付宝支付: " + amount);
    }
}

// 使用
PaymentStrategy strategy = new AlipayStrategy();
strategy.pay(100);
```

1. **八股文要熟**：拼多多一面主打八股文拷打

4. **心态要好**：面试强度大，要做好心理准备
5. **分布式要学**：微服务、Redis、消息队列都要了解
