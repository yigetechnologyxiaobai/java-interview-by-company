## 面试风格

**特点**：高压测试，笔试淘汰率高（80%+），一面八股文拷打

**轮次**：笔试 + 至少 3 轮技术面 + 1 轮 HR

**重点**：Java 基础、多线程、JVM、操作系统、分布式

**面试官画像**：高强度技术拷问，喜欢深挖底层原理

## 面试技巧

1. **笔试要认真准备**：淘汰率很高，笔试是第一关
2. **基础要扎实**：八股文拷打，要能抗住压力
3. **底层原理**：喜欢深挖，要理解原理
4. **抗压能力**：高强度面试，心态要好
