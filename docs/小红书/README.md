## 面试风格

**特点**：注重实战能力，关注高并发场景，喜欢项目经验

**轮次**：通常 2-3 轮技术面 + 1 轮 HR

**重点**：Redis、MySQL、分布式、高并发、业务场景设计

**面试官画像**：喜欢问场景题，关注实战能力
