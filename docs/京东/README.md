## 面试风格

**特点**：注重基础扎实，偏爱八股文，考察范围广

**轮次**：通常 2-3 轮技术面 + 1 轮 HR

**重点**：Java 基础、MySQL、Redis、Spring、并发编程

1. **重视基础**：京东偏爱八股文，要背熟

3. **保持谦虚**：不会就说不会
4. **注意礼貌**：面试官多为技术人员，态度要好
