## 58同城 Java 社招面经（2026-04）

### 一面真题

#### Q1: MySQL 迁移到 TiDB 做了哪些工作？

**参考答案**：

**迁移步骤**：
1. 评估数据量、性能指标
2. 兼容性测试（TiDB 兼容 MySQL 协议）
3. 数据同步（DM 工具）
4. 双写验证
5. 灰度切换

**关键问题**：
- 自增主键（TiDB 自增不连续）
- 性能 TP90/TP95 测试

---

#### Q2: TiDB 性能问题怎么调研？

**参考答案**：

**调研方法**：
1. 查看 TiDB Dashboard
2. 分析慢查询日志
3. 查看 Region 分布
4. 检查热点问题

---

#### Q3: Redis 分布式锁和红锁？

**参考答案**：

**Redlock 算法**：
- 在多个 Redis 节点同时加锁
- 超过半数节点成功才算成功
- 解决单节点故障问题

**问题**：
- 时钟漂移可能导致锁失效
- 网络延迟问题

---

#### Q4: SpringBean 创建过程？AOP 实现原理？

**参考答案**：

**Bean 创建流程**：
1. 实例化（createBeanInstance）
2. 属性填充（populateBean）
3. 初始化（initializeBean）
4. AOP 代理

**循环依赖解决**：
- 三级缓存
- 早期引用提前暴露

---

#### Q5: Spring 怎么解决循环依赖？

**参考答案**：

**三级缓存**：
- singletonObjects：完整 Bean
- earlySingletonObjects：早期引用
- singletonFactories：Bean 工厂

**流程**：
A 创建 → 注入 B → B 创建 → 注入 A（从三级缓存获取早期 A）→ B 完成 → A 完成

**为什么需要三级？**
二级缓存能解决循环依赖，三级是为了支持 AOP 代理。

---

#### Q6: JDK 动态代理和 CGlib 区别？

**参考答案**：

| 对比项 | JDK | CGlib |
|--------|-----|-------|
| 要求 | 实现接口 | 继承类 |
| 原理 | 反射 | 字节码增强 |
| 性能 | 较低 | 较高 |

---

#### Q7: AQS 原理？

**参考答案**：

**核心结构**：
- state（同步状态）
- CLH 队列（等待线程）

**加锁流程**：
1. CAS 尝试获取 state
2. 失败则入队等待
3. 前驱节点释放后唤醒
