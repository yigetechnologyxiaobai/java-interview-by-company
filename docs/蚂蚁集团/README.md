# 蚂蚁集团 面试题

## 面试风格

**特点**：项目深挖 + 业务场景 + 八股较少，关注 AI 方向

**轮次**：通常 2-3 轮技术面 + 1 轮 HR

**重点**：Java 基础、线程池、JVM、项目经验、AI 工程化

**面试官画像**：喜欢问项目架构和技术选型，AI/Agent 方向是热门

## 面试技巧

1. **项目要熟悉**：每个项目的架构、技术选型、难点要能说清楚
2. **AI 方向加分**：Agent、RAG 相关项目经验加分
3. **基础扎实**：线程池、JVM 等基础要能讲清楚
4. **业务场景**：能说出具体业务场景的设计方案
