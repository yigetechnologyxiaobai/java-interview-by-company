#### Q1: SpringBoot 的 IOC 和 AOP？

**参考答案**：

**IOC（控制反转）**：
- 将对象的创建和管理交给 Spring 容器
- 通过依赖注入（DI）实现解耦

**IOC 容器核心**：
```java
// Bean 的生命周期
1. 实例化（Instantiation）
2. 属性赋值（Populate）
3. 初始化（Initialization）
4. 销毁（Destruction）

// 核心接口
BeanFactory：基础容器
ApplicationContext：高级容器（国际化、事件发布等）
```

**AOP（面向切面编程）**：
- 将公共逻辑抽取为切面
- 通过动态代理实现

**AOP 核心概念**：
```java
@Aspect
@Component
public class LoggingAspect {
    
    @Pointcut("execution(* com.example.service.*.*(..))")
    public void serviceMethods() {}
    
    @Before("serviceMethods()")
    public void logBefore(JoinPoint joinPoint) {
        System.out.println("执行前: " + joinPoint.getSignature());
    }
    
    @AfterReturning("serviceMethods()")
    public void logAfter(JoinPoint joinPoint) {
        System.out.println("执行后: " + joinPoint.getSignature());
    }
}
```

#### Q2: SpringBoot 自动装配原理？

**参考答案**：

**核心注解**：
```java
@SpringBootApplication = 
    @SpringBootConfiguration    // 配置类
    + @EnableAutoConfiguration  // 自动装配
    + @ComponentScan            // 扫描组件
```

**自动装配流程**：
1. `@EnableAutoConfiguration` 导入 `AutoConfigurationImportSelector`
2. 通过 `SpringFactoriesLoader` 加载 `META-INF/spring.factories`
3. 根据 `@Conditional` 条件判断是否装配

**条件注解**：
```java
@ConditionalOnClass           // 存在某个类
@ConditionalOnMissingBean     // 不存在某个 Bean
@ConditionalOnProperty        // 配置属性满足条件
@ConditionalOnWebApplication  // Web 应用环境
```

**示例**：
```java
@Configuration
@ConditionalOnClass(DataSource.class)
@EnableConfigurationProperties(DataSourceProperties.class)
public class DataSourceAutoConfiguration {
    
    @Bean
    @ConditionalOnMissingBean
    public DataSource dataSource(DataSourceProperties properties) {
        return properties.initializeDataSourceBuilder().build();
    }
}
```

#### Q4: MySQL 分表策略？优缺点？

**参考答案**：

**分表策略**：

| 策略 | 说明 | 适用场景 |
|------|------|---------|
| 范围分表 | 按时间/ID范围 | 日志、订单 |
| Hash分表 | 按Hash值分 | 用户数据 |
| 一致性Hash | 解决扩容问题 | 大规模数据 |

**优点**：
- 单表数据量减少，查询效率提高
- 分散IO，提高并发能力

**缺点**：
- 跨表查询复杂
- 数据迁移困难
- 分布式事务问题

**代码示例**：
```java
// 分表路由
public String getTableName(Long userId) {
    int tableIndex = userId % 8;
    return "user_" + tableIndex;
}

// 分库分表
public String getDbAndTable(Long userId) {
    int dbIndex = userId / 8 % 4;  // 4个库
    int tableIndex = userId % 8;   // 每库8张表
    return "db_" + dbIndex + ".user_" + tableIndex;
}
```

#### Q6: 算法题 - 返回数组中子数组范围的和

**参考答案**：

```java
public long subArrayRanges(int[] nums) {
    long sum = 0;
    int n = nums.length;
    
    for (int i = 0; i < n; i++) {
        int min = nums[i], max = nums[i];
        for (int j = i; j < n; j++) {
            min = Math.min(min, nums[j]);
            max = Math.max(max, nums[j]);
            sum += max - min;
        }
    }
    
    return sum;
}
```

**时间复杂度**：O(n²)

#### Q2: 算法题 - 二分查找

**参考答案**：

```java
public int binarySearch(int[] nums, int target) {
    int left = 0, right = nums.length - 1;
    
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) {
            return mid;
        } else if (nums[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    
    return -1;
}
```

#### Q4: MySQL 什么时候不走索引？

**参考答案**：

**索引失效场景**：
1. 在索引列上使用函数
2. 隐式类型转换
3. 使用 `!=` 或 `<>`
4. 使用 `OR` 连接非索引列
5. `LIKE` 以 `%` 开头
6. 索引列参与计算

**示例**：
```sql
-- 失效
SELECT * FROM user WHERE YEAR(create_time) = 2026;
SELECT * FROM user WHERE name LIKE '%张';
SELECT * FROM user WHERE id + 1 = 100;
SELECT * FROM user WHERE id != 100;

-- 有效
SELECT * FROM user WHERE create_time >= '2026-01-01';
SELECT * FROM user WHERE name LIKE '张%';
SELECT * FROM user WHERE id = 100 - 1;
```

#### Q5: 算法题 - 除自身以外数组的乘积

**参考答案**：

```java
public int[] productExceptSelf(int[] nums) {
    int n = nums.length;
    int[] result = new int[n];
    
    // 左侧乘积
    result[0] = 1;
    for (int i = 1; i < n; i++) {
        result[i] = result[i - 1] * nums[i - 1];
    }
    
    // 右侧乘积
    int right = 1;
    for (int i = n - 1; i >= 0; i--) {
        result[i] *= right;
        right *= nums[i];
    }
    
    return result;
}
```

**时间复杂度**：O(n)
**空间复杂度**：O(1)（输出数组不计）

**参考答案**：

```java
public int[] productExceptSelf(int[] nums) {
    int n = nums.length;
    int[] result = new int[n];
    
    // 前缀乘积
    result[0] = 1;
    for (int i = 1; i < n; i++) {
        result[i] = result[i - 1] * nums[i - 1];
    }
    
    // 后缀乘积
    int suffix = 1;
    for (int i = n - 1; i >= 0; i--) {
        result[i] *= suffix;
        suffix *= nums[i];
    }
    
    return result;
}
```

#### Q4: 算法题 - 手撕 LRU

**参考答案**：

```java
class LRUCache {
    private int capacity;
    private Map<Integer, Node> cache;
    private Node head, tail;
    
    public LRUCache(int capacity) {
        this.capacity = capacity;
        cache = new HashMap<>();
        head = new Node();
        tail = new Node();
        head.next = tail;
        tail.prev = head;
    }
    
    public int get(int key) {
        Node node = cache.get(key);
        if (node == null) return -1;
        moveToHead(node);
        return node.value;
    }
    
    public void put(int key, int value) {
        Node node = cache.get(key);
        if (node != null) {
            node.value = value;
            moveToHead(node);
        } else {
            node = new Node(key, value);
            cache.put(key, node);
            addToHead(node);
            if (cache.size() > capacity) {
                Node removed = removeTail();
                cache.remove(removed.key);
            }
        }
    }
    
    private void moveToHead(Node node) {
        removeNode(node);
        addToHead(node);
    }
    
    private void addToHead(Node node) {
        node.prev = head;
        node.next = head.next;
        head.next.prev = node;
        head.next = node;
    }
    
    private void removeNode(Node node) {
        node.prev.next = node.next;
        node.next.prev = node.prev;
    }
    
    private Node removeTail() {
        Node node = tail.prev;
        removeNode(node);
        return node;
    }
    
    class Node {
        int key, value;
        Node prev, next;
        Node() {}
        Node(int key, int value) {
            this.key = key;
            this.value = value;
        }
    }
}
```

### Q1: 转转 - 算法：字符串 AB 最小步数移动

**题目**：一个字符串，只包含 A、B 两个字符，求把所有 A 移动到左边，所有 B 移动到右边的最小步数

**参考答案**：

```java
public int minSteps(String s) {
    int countA = 0;
    for (char c : s.toCharArray()) {
        if (c == 'A') countA++;
    }
    
    // 滑动窗口找包含最多 A 的长度为 countA 的窗口
    int maxAInWindow = 0;
    int currentA = 0;
    
    for (int i = 0; i < s.length(); i++) {
        if (s.charAt(i) == 'A') currentA++;
        if (i >= countA && s.charAt(i - countA) == 'A') currentA--;
        if (i >= countA - 1) maxAInWindow = Math.max(maxAInWindow, currentA);
    }
    
    return countA - maxAInWindow;
}
```

**思路**：找到包含最多 A 的窗口，窗口外的 A 需要移动
