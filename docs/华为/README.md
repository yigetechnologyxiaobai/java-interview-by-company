# 华为 面试题

## 面试风格

**特点**：注重综合素质，技术深度和广度并重，重视项目经验

**轮次**：通常 2-3 轮技术面 + 1 轮综合面 + 1 轮 HR

**重点**：并发编程、JVM、分布式、数据库、系统设计

## 八、项目经验


华为非常重视项目经验，常见追问：

1. 项目难点是什么？
2. 为什么选择这个技术方案？
3. 有没有考虑其他方案？
4. 遇到什么问题？怎么解决的？
5. 系统的 QPS、RT 指标？

---


## 九、面试技巧


1. **技术深度**：华为注重深度，要知其然知其所以然
2. **系统思维**：要有全局视角，考虑边界情况
3. **项目准备**：项目细节要清楚，难点要能说透
4. **综合素质**：华为看重综合素质，沟通表达很重要
5. **技术热情**：展示对技术的热情和学习能力
