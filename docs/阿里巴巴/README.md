## 项目深挖


**常见追问**：

1. 这个项目的难点是什么？
2. 为什么选择这个技术方案？
3. 有没有考虑其他方案？对比分析？
4. 遇到过什么问题？怎么解决的？
5. 如果让你重新设计，会怎么改进？
6. 系统的瓶颈在哪里？如何优化？

**回答框架**：

```
项目背景 → 问题分析 → 方案对比 → 最终选择 → 遇到的问题 → 解决方案 → 效果评估
```

---


## 面试技巧


1. **不要只回答是什么，要回答为什么**
2. **准备好项目的每个细节**
3. **不懂就说不懂，不要乱编**
4. **准备好反问环节**
5. **展示学习能力**：最近在看什么书/文章？
