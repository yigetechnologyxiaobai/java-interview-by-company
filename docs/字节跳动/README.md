# 字节跳动 面试题

## 面试风格

**特点**：算法驱动，系统设计占比大，代码能力要求高

**轮次**：通常 3-4 轮技术面（每轮都有算法）

**重点**：算法、系统设计、基础

**面试官画像**：喜欢让候选人在白板/在线编辑器写代码，代码风格、边界处理都很重要

## 面试技巧

1. **算法必练**：每轮都有算法，LeetCode 要刷熟练
2. **代码规范**：代码风格、边界处理、命名都很重要
3. **系统设计**：关注架构设计能力
4. **手写代码**：白板/在线编辑器写代码，要练习
