## 项目深挖


美团非常注重项目经验，常见追问：

1. **项目难点是什么？**
2. **为什么选择这个方案？**
3. **有没有考虑其他方案？**
4. **遇到什么问题？怎么解决的？**
5. **如果重新设计，会怎么改进？**
6. **系统的 QPS 是多少？**
7. **如何做性能优化？**

---


## 面试技巧


1. **重视实战**：美团偏爱场景题，准备项目细节
2. **数据说话**：系统的 QPS、RT 等指标要清楚
3. **方案对比**：回答设计题时多给几种方案对比
4. **关注业务**：美团业务驱动，理解业务很重要
5. **保持谦虚**：不懂就说不懂，不要强行解释
