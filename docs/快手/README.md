# 快手 面试题

## 面试风格

**特点**：一面偏基础八股，二面深挖项目和算法（Hard 较多）

**轮次**：通常 2-3 轮技术面 + 1 轮 HR

**重点**：JUC 多线程、并发特性、网络编程、算法

**面试官画像**：一面常规八股，二面深挖项目，手撕 Hard 级别算法

## 面试技巧

1. **一面基础要扎实**：快手一面主打八股文，常规问题要熟练
2. **二面项目要深挖**：项目难点、技术选型要能说清楚
3. **算法要多练**：快手喜欢出 Hard，手撕能力要强
